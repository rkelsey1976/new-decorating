Before & After images (gallery page + local template)
=====================================================

GALLERY PAGE (this folder):
  The gallery page "Before & After" section uses images from this folder.
  Add these files for the railings slideshow:
    before-1.png   → main "Before" panel
    before-2.png   → grid
    before-3.png   → grid
    after-1.png    → main "After" panel
    after-2.png    → grid

LOCAL TEMPLATE (screenshot for GBP posts):
  For the HTML template, use: website/tools/images/before-after/
  Put your photos there and open website/tools/before-after-template.html
  in your browser to take a screenshot for Google Business Profile.
